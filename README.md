# 🚀 Pratik Kumar Verman | Portfolio

Welcome to my personal portfolio repository. This site is a live demonstration of my work as an aspiring DevOps Engineer, focusing on infrastructure automation, CI/CD, and cloud-native workflows.

---

## 🛠️ Tech Stack
*   **Framework:** Next.js 16 (Turbopack)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **Deployment:** Vercel

## 🏗️ DevOps Lab Focus
I am currently focused on building and documenting end-to-end DevOps infrastructure:
*   **CI/CD:** Jenkins & GitHub Actions automation.
*   **Containerization:** Docker-based development workflows.
*   **GitOps:** Argo CD deployments on Kubernetes.
*   **IaC:** Provisioning repeatable environments with Terraform.
